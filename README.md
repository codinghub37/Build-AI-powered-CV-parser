# 📄 AI-Powered CV Parser & Resume Analyzer

## Overview

AI-Powered CV Parser is a Streamlit-based application that uses Google's Gemini AI model to automatically analyze and extract structured information from resume PDFs.

The system converts uploaded resume documents into machine-readable JSON format by identifying key candidate details such as personal information, education, work experience, certifications, projects, skills, and languages.

This project demonstrates the practical application of Artificial Intelligence, Natural Language Processing (NLP), and Large Language Models (LLMs) in recruitment and HR automation.

## How It Works

1. User uploads a resume PDF.
2. PDF pages are converted into images.
3. Images are sent to Google's Gemini AI model.
4. Gemini analyzes the resume content.
5. Structured candidate information is extracted.
6. Results are displayed in JSON format.
7. Parsed data can be downloaded as a JSON file.
